EasyForm API

The EasyForm class provides a fluent interface for building HTML forms with validation, AJAX support, and responsive design.

Constructor

__construct() Constructor

Creates a new EasyForm instance with optional configuration.

public function __construct(string $id = 'easyform', array $config = []): EasyForm

Parameters

Parameter Type Required Default Description
$id string Optional 'easyform' Unique form identifier
$config array Optional [] Configuration options
Returns: EasyForm - The form instance for method chaining

Example

'bootstrap', 'size' => 'large', 'width' => 800, 'language' => 'en' ]); ?>

Field Methods

text() Field

Creates a text input field.

public function text(string $name, string $label = '', array $options = []): self

Parameters

Parameter Type Required Description
$name string Required Field name attribute
$label string Optional Field label text
$options array Optional Field configuration options

Available Options

  • required (bool): Make field mandatory
  • placeholder (string): Placeholder text
  • value (string): Default value
  • maxlength (int): Maximum character length
  • readonly (bool): Make field read-only
  • disabled (bool): Disable field
  • icon (string): Field icon
  • help (string): Help text
  • class (string): Additional CSS classes

Examples

text('username', 'Username'); // Required field with placeholder $form->text('name', 'Full Name', [ 'required' => true, 'placeholder' => 'Enter your full name' ]); // Advanced text field $form->text('title', 'Job Title', [ 'value' => 'Software Developer', 'maxlength' => 100, 'icon' => 'briefcase', 'help' => 'Your current or desired job title' ]); ?>
email() Field

Creates an email input field with built-in email validation.

public function email(string $name, string $label = '', array $options = []): self

Example

email('email', 'Email Address', [ 'required' => true, 'placeholder' => 'your@email.com', 'icon' => 'envelope' ]); ?>
select() Field

Creates a select dropdown field with options.

public function select(string $name, string $label, array $options, array $config = []): self

Parameters

Parameter Type Required Description
$name string Required Field name
$label string Required Field label
$options array Required Select options (value => label)
$config array Optional Field configuration

Examples

select('country', 'Country', [ 'us' => 'United States', 'uk' => 'United Kingdom', 'ca' => 'Canada' ]); // Select with default value $form->select('priority', 'Priority', [ 'low' => 'Low', 'medium' => 'Medium', 'high' => 'High' ], ['value' => 'medium']); // Multiple select $form->select('skills', 'Skills', [ 'php' => 'PHP', 'javascript' => 'JavaScript', 'python' => 'Python' ], ['multiple' => true]); ?>

Layout Methods

group() Layout

Creates a field group (tab, accordion, or fieldset).

public function group(string $name, string $label, array $options = []): self

Group Types

  • tab: Tabbed interface
  • accordion: Collapsible sections
  • fieldset: HTML fieldset with legend

Example

group('personal', 'Personal Information', [ 'type' => 'tab', 'active' => true ]) ->text('name', 'Name') ->email('email', 'Email') ->endGroup(); ?>

Validation Methods

rule() Validation

Adds validation rules to a field.

public function rule(string $field, string|array $rules): self

Available Rules

  • required: Field must not be empty
  • email: Valid email address
  • url: Valid URL
  • numeric: Numeric value
  • minlength[n]: Minimum length
  • maxlength[n]: Maximum length
  • matches[field]: Match another field

Examples

rule('email', 'required|email'); // Multiple rules $form->rule('password', ['required', 'minlength[8]']); // Password confirmation $form->rule('confirm_password', 'required|matches[password]'); ?>

Output Methods

display() Output

Outputs the form HTML directly to the browser.

public function display(): void

Example

text('name', 'Name') ->submit('Submit') ->display(); ?>
render() Output

Returns the form HTML as a string.

public function render(): string
Returns: string - Complete form HTML

Example

render(); echo $html; // Or use in templates $template = str_replace('{{form}}', $form->render(), $template); ?>

EasyList API

The EasyList class creates powerful data tables with search, sorting, pagination, and export capabilities.

Data Methods

data() Data

Sets the data source for the table.

public function data(array $data): self

Parameters

Parameter Type Required Description
$data array Required Array of associative arrays (rows)

Example

1, 'name' => 'John', 'email' => 'john@example.com'], ['id' => 2, 'name' => 'Jane', 'email' => 'jane@example.com'] ]; $list = new EasyList('users'); $list->data($users); ?>

Column Methods

column() Column

Defines a table column with options.

public function column(string $key, string $label, array $options = []): self

Parameters

Parameter Type Required Description
$key string Required Data key/field name
$label string Required Column header text
$options array Optional Column configuration

Column Options

  • sortable (bool): Enable sorting
  • searchable (bool): Include in search
  • width (string): Column width
  • align (string): Text alignment
  • type (string): Data type for formatting
  • format (string): Custom format string
  • template (string|callable): Custom template

Examples

column('name', 'Full Name'); // Sortable column $list->column('email', 'Email', ['sortable' => true]); // Formatted column $list->column('created_at', 'Created', [ 'type' => 'date', 'format' => 'M j, Y' ]); // Custom template $list->column('status', 'Status', [ 'template' => function($row) { $class = $row['active'] ? 'green' : 'red'; return "{$row['status']}"; } ]); ?>

Feature Methods

searchable() Feature

Enables global search functionality.

public function searchable(array $options = []): self

Example

searchable([ 'placeholder' => 'Search users...', 'debounce' => 300 ]); ?>
paginate() Feature

Enables pagination with specified page size.

public function paginate(int $pageSize = 25): self

Example

paginate(50); // 50 items per page ?>
exportable() Feature

Enables data export in specified formats.

public function exportable(array $formats = ['csv', 'excel']): self

Supported Formats

  • csv: Comma-separated values
  • excel: Microsoft Excel
  • json: JSON format
  • pdf: PDF document

Example

exportable(['csv', 'excel', 'pdf']); ?>

Display Methods

display() Output

Outputs the table HTML directly.

public function display(): void

Example

data($users) ->column('name', 'Name') ->column('email', 'Email') ->searchable() ->display(); ?>
